A beautiful frontend website switcher widget for visitors to navigate between multiple websites, perfect for headers and navigation areas.

== Description ==

The Website Switcher block provides an elegant frontend solution for visitors to navigate between multiple websites or web properties. Add it to your header, navigation, or any frontend area to give your visitors quick access to related sites.

Designed specifically for frontend use, this block displays a customizable icon that, when clicked by visitors, reveals a dropdown list of websites complete with their own icons and links.

Perfect for:
* Multi-site networks visible to visitors
* Agency portfolios with multiple client sites
* Product families with separate websites
* Brand ecosystems
* International site versions
* Corporate site networks

Features:
* Frontend-focused design for site visitors
* Customizable switcher icon
* Add unlimited websites to the list
* Each website can have its own icon and URL
* Fully customizable from the block editor
* Clean, modern design optimized for headers
* Smooth animations and transitions
* Responsive layout for all devices
* Accessible keyboard navigation
* Click-outside to close functionality
* Works perfectly in FSE headers and templates

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/website-switcher` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the Website Switcher block to your header, navigation, or any page through the block editor or Site Editor
4. Configure your websites list using the block inspector panel
5. The switcher will now be visible to all site visitors on the frontend

== Frequently Asked Questions ==

= How do I add this to my site header? =

You can add the Website Switcher block to your header in several ways:
1. Using the Site Editor (Full Site Editing): Edit your header template and add the block
2. Using a page builder: Add the block to your header area
3. Using widgets: If your theme supports header widgets, add the block there
4. Directly in posts/pages: Simply add the block where you want it to appear

= How do I add websites to the switcher? =

Select the block in the editor, then use the inspector panel on the right to add, edit, or remove websites from your list. Each website requires a name and URL, and you can optionally add an icon URL.

= Will visitors see this on the frontend? =

Yes! This block is specifically designed for frontend use. Once added to your site, all visitors will be able to see and interact with it.

= Can I customize the appearance to match my theme? =

Yes! The block comes with clean default styling that works well with most themes, but you can add custom CSS to match your site's design perfectly. Use the block's CSS classes to target specific elements.

= Does it work on mobile devices? =

Absolutely! The website switcher is fully responsive and works great on all device sizes, including phones and tablets.

= Can I use custom icons? =

Yes, you can provide URLs to any image to use as icons for both the main switcher button and individual websites in the list.

= Does it support keyboard navigation? =

Yes, the switcher is fully accessible with keyboard support, including Escape key to close the dropdown.
